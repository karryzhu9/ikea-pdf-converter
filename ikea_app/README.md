# IKEA Planner PDF → Excel Converter

Upload any IKEA Planner PDF and instantly get a clean Excel file with:
- **Sheet 1 (All Items):** Every item with product name, description, article number, quantity, unit price, total price
- **Sheet 2 (Selected Items):** Paste your highlighted items here to build a quotation

## Deploy to Streamlit Cloud (Free)

1. Fork or push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repo
4. Set main file path: `app.py`
5. Click Deploy — done, you get a free public URL

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
